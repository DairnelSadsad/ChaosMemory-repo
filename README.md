# Chaos Memory

## Project Descrpition 

Chaos Memory is a fast-paced memory matching card game with a chaotic twist. 
Players flip cards to find matching pairs while managing lives and earning points. 
Unlike traditional memory games, special cards add unpredictable effects: 
Peek cards briefly reveal hidden cards, Heart cards restore lives, and Tornado cards rotate the board to confuse players. 
Each level begins with a short preview of all cards, challenging players to memorize positions before the game starts. 
As difficulty increases, more cards and pairs are added, making the game more intense and strategic.
